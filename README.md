# Customer Support Chat System

The **Customer Support Chat System** is a web application that enables real-time communication between customers and support agents. It features secure user authentication, role-based access, chat history management, and a visually appealing, responsive UI.

## Features
- Real-time chat between customers and support agents.
- Role-based authentication using JWT (JSON Web Tokens).
- Manage customer queries and agent responses.
- View chat histories for previous conversations.
- Responsive and user-friendly UI for enhanced usability.

## Installation and Setup
Follow these steps to set up the project locally:

### Backend
1. Navigate to the `backend` folder:
   ```bash
   cd backend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

### Frontend
1. Navigate to the `frontend` folder:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

## Tools and Technologies Used
- **Frontend**: React.js, Tailwind CSS
- **Backend**: Node.js, Express.js
- **Authentication**: JWT (JSON Web Tokens)
- **Database**: Relational Database (PostgreSQL)

Note Replace your database url from .env file
